Appendix 4. IBM JDKの注意点

IBM JDK (WebSphereTM 付属など) を使用していて、かつ下記の条件を満たすとき、テンプレートを読み込む際に sun.io.MalformedInputException が発生します。

  • HTML ファイルに UTF-8 以外の文字コードを使用している。
  • HTML ファイルの先頭 2048 バイト以内に ascii 文字以外がある。(正確には UTF-8 として読んだときに不正なバイトがある)

対応方法

5-1「エンジン設定方法」に記載された方法で TemplateBuilder のパラメータ "defaultCharset" に適切な文字コードを指定してください。
