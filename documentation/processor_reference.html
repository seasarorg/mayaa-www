appendix B. Maya core プロセッサ リファレンス

write プロセッサ

オブジェクトを文字列として出力します。

属性名説明必須値の制限
value出力する値。○

if プロセッサ

タグのボディが有効です。

test の値が true の場合のみボディを処理します。

属性名説明必須値の制限
test判定する条件式。true ならば有効。○論理値

for プロセッサ

タグのボディが有効です。

繰り返し処理をします。最初に一度 init を実行してループ処理を開始します。1 回の処理の最初に test を実行し、true ならループ処理を続行し、false ならループ処理を終了します。1 回の処理の最後に after を実行します。

ループ回数の上限を指定している場合、ループ回数が上限数以上になると例外が発生します。(org.seasar.maya.impl.engine.processor.TooManyLoopException)

属性名説明必須値の制限
init初期化処理。○ 
test判定する条件式。true ならば処理を続行。○論理値
afterループ処理 1 回の後処理。○ 
maxループ回数の上限。 整数値

with プロセッサ

タグのボディが有効です。

with プロセッサ自体は何もしません。複数のプロセッサをグルーピングするために使います。

属性名説明必須値の制限
script実行するスクリプト。

encodingsrc で指定したファイルを読み込むときの文字エンコーディング Java で利用できる文字セット名

doBody プロセッサ

割り当てられたタグのボディを処理します。

insert プロセッサ

コンポーネントを使う場合

doRender プロセッサ

コンポーネントを定義する場合

割り当てられたタグをコンポーネントとして定義します。名前は同じテンプレート上で重複しないように付けてください。

属性名説明必須値の制限
nameコンポーネントの名前。デフォルトは "" (空文字列)。 文字列

レイアウトを利用する場合

レイアウトの insert に渡す部分を定義します。

属性名説明必須値の制限
nameレイアウトの insert 名。デフォルトは "" (空文字列)。 文字列

doBase プロセッサ

コンポーネントで使う場合

コンポーネントを使う元テンプレートのタグのボディを処理します。

レイアウトを利用するテンプレートで使う場合

レイアウトの insert のタグのボディを処理します。

null プロセッサ

何も出力しません。

element プロセッサ

タグのボディが有効です。

name で指定したタグを出力します。m:element タグのボディに attribute プロセッサを書くと、属性を追加できます。

属性名説明必須値の制限
name生成するタグの要素名。○文字列

attribute プロセッサ

親タグのプロセッサに属性を追加します。暗黙の doBody の条件において、唯一例外として空白同様の扱いを受けるプロセッサです。

属性名説明必須値の制限
name生成する属性名。○文字列
value生成する属性値。○文字列

echo プロセッサ

タグのボディが有効です。

割り当てられたタグをそのまま出力します。m:echo タグのボディに attribute プロセッサを書くと、属性を追加または変更できます。

ignore プロセッサ

割り当てた id を処理対象外にします。(正確には ignore はプロセッサではなくエンジン機能です)
