appendix D. JavaScript を書くときの Tips

Java のクラスを使う

Java のクラスは、java.lang.String や java.util.ArrayList のようにパッケージが "java." で始まるクラスであれば、完全修飾名をそのまま書くだけで利用できます。しかしそれ以外のパッケージ ("org." や "com." など) の場合、先頭に "Packages." を付ける必要があります。

例:
var list = new java.util.ArrayList();
var myClass = new Packages.org.example.MyClass();

例外を処理する

JavaScript にも try 〜 catch 構文はありますので、Java で発生した例外を JavaScript で処理することもできます。Java でアプリケーションを作るときには Java 内で処理するものでしょうが、デバッグ時には役立つかもしれません。

var catched = "";
try {
    var list = myLogic.findByName(name);
} catch (e) {
    catched = e;
}

<m:write id="error" value="${ catched }" />
